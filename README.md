# Homebrew Channel Archive
This branch contains WADs of (nearly) all public Homebrew Channel channels that I've dumped.

![image](https://user-images.githubusercontent.com/71722170/141535470-bc64165e-259b-4dee-ad99-cfbfddcc95b6.png)

unpatched = Unmodified, simply dumped.
patched = Changed channel name and title id (can be used on 4.3/Wii Mini/vWii without Priiloader).
patched old = Patched but old, do not recommend using.
